# Phase 02 – Resource Group Deployment

## Objective
Deploy the initial Azure Resource Group using Terraform as the foundation for the 3-tier architecture lab.

## Files Used
- versions.tf
- providers.tf
- variables.tf
- main.tf
- outputs.tf

## Terraform Actions
- Initialized Terraform
- Validated Azure authentication
- Ran `terraform plan`
- Ran `terraform apply`

## Deployed Resource
- Resource Group: `rg-azure-3tier-lab`
- Region: `westeurope`

## Validation
The resource group was verified successfully in the Azure Portal.

## Outcome
This phase established the Azure landing point for the rest of the infrastructure build.
