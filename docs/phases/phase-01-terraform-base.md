# Phase 01 – Terraform Base Configuration

## Objective
Prepare the Terraform project structure and base configuration for Azure infrastructure deployment.

## Files Created
- `versions.tf`
- `providers.tf`
- `variables.tf`
- `main.tf`
- `outputs.tf`

## Configuration Details
- Defined required Terraform version
- Configured AzureRM provider
- Declared reusable variables for resource group name and location
- Added initial resource definition structure
- Added outputs for validation

## Terraform Foundation
This phase established the Infrastructure as Code foundation for the full lab and prepared the environment for repeatable Azure deployments.

## Outcome
The Terraform project structure was successfully prepared and ready for resource deployment in later phases.
